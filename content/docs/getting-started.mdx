# Getting Started
Welcome to the project! This guide helps you get up and running quickly with the docs site.

## What’s included
- A simple documentation structure in `content/docs`
- Markdown and MDX pages for easy content creation
- A lightweight site layout for getting started fast

## Install dependencies
First, install the project dependencies:

```bash
npm install
```

## Run the development server
Start the local site and preview your documentation:

```bash
npm run dev
```

Then open the URL shown in your terminal, usually `http://localhost:3000`.

## Add a new page
Create a new file in `content/docs`, for example:

```bash
touch content/docs/new-guide.mdx
```

Then add a title and content using standard Markdown or MDX:

```mdx
---
title: New Guide
---

# New Guide

Write your content here.
```

## Next steps
- Edit `content/docs/index.mdx` to update the home page
- Add additional guides under `content/docs`
- Use MDX to include React components inside your docs if the site supports it

Happy writing!
